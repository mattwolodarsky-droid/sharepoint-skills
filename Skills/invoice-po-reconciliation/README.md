# Invoice to Purchase Order Reconciliation with SharePoint AI Skills

This demo shows reconciliation of one or more invoices with corresponding purchase order. One of the most common use case within the procue to purchase scenario is to ensure that invoices are compliant with purchase order. If the items in the invoice do not match with items in the purchase order, or if the invoice total exceeds the purchase order total, the invoice is deemedd as non compliant.

The process of reconciliation is conducted manually or through an automation that includes low code and pro code approach. In this demonstration I will show you how can now achieve reconciliation using the Skills without writing a single line of code

# RFP Response - Extraction

Parses an RFP document stored in SharePoint and populates a structured tracking list with every question, mandatory requirement, submission rule, evaluation criterion, and administrative certification — categorized, source-traced, and ready for owner assignment.

![preview](./assets/preview.png)

## What you get

- Populated RFP Tracker SharePoint List — one row per extracted item
- Items categorized across 5 types (Question, Mandatory, Submission Rule, Evaluation Criterion, Certification)
- Mandatory flag and source location on every row
- Addenda applied correctly — later versions override earlier ones
- Summary: item counts by category, missing attachments, ambiguous items flagged

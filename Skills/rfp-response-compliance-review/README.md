# RFP Response - Compliance Review

Checks a proposal draft against the firm's configurable compliance rules — prohibited language, required disclosures, role and positioning boundaries, and procurement submission requirements — logging all findings and blocking publication until High severity issues are resolved.

![preview](./assets/preview.png)

## What you get

- Findings logged to the Review Findings SharePoint List — severity, location, verbatim text, recommended fix
- Draft annotated inline at each finding's location
- Summary: findings by severity, all High findings listed, page count vs. RFP limit
- High severity findings block publication until resolved by an authorized approver
- Links to the annotated draft and filtered Review Findings list

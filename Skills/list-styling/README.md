# List Styling Engine

Transform SharePoint lists and document libraries into fully art-directed views using column formatting, view formatting, row templates, and tile layouts. Works with any style token file in this collection to produce "I can't believe that's SharePoint" results.

## What you get

- A step-by-step workflow for mapping your list's columns to a style token file
- Complete `rowFormatter` JSON that replaces the default column grid with a fully designed card layout
- Per-column formatters for status badges, progress bars, and deadline compositions
- Instructions for creating a custom view so the formatting never touches your default "All Items" view
